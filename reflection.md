Reflection on Working with AI

For this assignment, I worked on scraping unstructured data from a real website and turning it into structured data using Python. At first, I didn’t fully understand what “unstructured data” really meant. Once I started looking at the HTML of the webpage, I realized how messy and complex it actually is behind the scenes. It is very different from working with a clean CSV file.

One of the most challenging parts was understanding how BeautifulSoup interacts with the HTML. I had to inspect the webpage, identify the correct tags, and experiment with different methods like find_all(). Sometimes the output was not what I expected, and I had to go back and adjust the code. That process was frustrating at times, especially when small mistakes caused errors. However, it helped me understand the logic more deeply.

Using an AI coding assistant helped me structure my approach and understand how the different libraries (requests, BeautifulSoup, pandas) work together. It made it easier to see the big picture of the scraping process. However, AI did not automatically solve everything. I still had to run the code, debug errors, and make sure I understood each step before moving forward. There were moments where I had to re-read the lecture notes and compare them to my code to make sure I wasn’t just copying without understanding.

I think the most important thing I learned is the difference between unstructured and structured data. Websites are written in HTML, which is not organized like a spreadsheet. Scraping allows us to extract specific pieces of information and reorganize them into a structured format like a DataFrame. That transformation is what makes the data usable for analysis.

Overall, I did enjoy the assignment, even though it was challenging. It felt more practical and closer to real-world data work compared to previous assignments. It also made me more comfortable reading HTML and understanding how web pages are built. This assignment showed me how powerful Python can be when working with online data, and it gave me more confidence in using external tools while still thinking critically about what the code is doing.
